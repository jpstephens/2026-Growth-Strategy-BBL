# BlackBridge Logistics 2026 Growth Strategy

A comprehensive interactive presentation website showcasing BlackBridge Logistics' strategic growth plan from $12K to $100K monthly net profit.

## Features

- **16 Comprehensive Sections** covering the complete growth strategy
- **6+ Data Visualizations** using Recharts for clear insights
- **Interactive Navigation** with smooth scrolling and progress tracking
- **Responsive Design** optimized for desktop, tablet, and mobile
- **Real Data** from 880 loads of historical transaction analysis
- **Professional Styling** aligned with BlackBridge brand colors and typography

## Tech Stack

- **React 19** with TypeScript for type safety
- **Vite** for fast development and optimized builds
- **Tailwind CSS** for utility-first styling
- **Recharts** for data visualization
- **Framer Motion** for smooth animations

## Project Structure

```
src/
├── components/
│   ├── layout/        # Header, Navigation, Footer
│   ├── content/       # Reusable components (Card, Table, Callout)
│   ├── charts/        # Data visualizations
│   └── sections/      # 16 main content sections
├── data/              # TypeScript data structures
├── hooks/             # Custom React hooks
└── styles/            # Global styles
```
