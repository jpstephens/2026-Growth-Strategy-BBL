## How to Log a Call (EVERY call!)

1. **Search** contact name in top bar
2. **Click** the contact
3. **Click** phone icon → **Log call**
4. **Fill in:**
   - Outcome: Connected / Left voicemail / No answer
   - Duration: Enter minutes
   - Notes: Add if real conversation
5. **Save**

**Remember:** No log = No dial count!

## Pipeline Stages & When to Move

| Stage | Move Here When... |
|-------|-------------------|
| **New Lead** | Just identified, no contact yet |
| **Contacted** | Made first call/email |
| **Engaged** | Had real 2-way conversation |
| **Qualified** | Has freight + authority + timeline |
| **Quote Sent** | Pricing delivered → **ENTER AMOUNT** |
| **Negotiating** | Discussing terms/rates |
| **First Load Booked** | Trial shipment scheduled |
| **Won** | Repeat business confirmed |
| **Lost** | Didn't convert → **SELECT REASON** |

## Required Fields

### At Quote Sent Stage:
**Amount** = Estimated annual value (e.g., $50,000)

### At Lost Stage:
**Lost Reason** = Select one:
- Price too high
- Went with competitor
- No decision / went dark
- Timing not right
- Bad fit / wrong target
- Service concerns
